Application Overview

As the name suggests, Lol in One is a Windows Phone application designed to improve the user's knowledge and skills when playing League of Legends. In order to achieve this, we are gathering information about the game itself, as well as from the current active players and display it all in one single user-friendly application, for everyone to have easy access to.

One key feature this application provides is that all the data displayed (champions, items, runes, masteries, builds) is UP TO DATE. Moreover, the user can refresh the database at any given moment, which automatically downloads the most recent data from the League of Legends API.

LoL in One Application Content

Lol in One is build from three main sections: Champions, Items and Summoner. The first two sections contain Static Data, which is constantly updated from the League of Legends API. The Summoner section alows the user to store a list of favorite summoners, keep track of their profile and statistics and also gain access to their current active game.
